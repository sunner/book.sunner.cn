第25页-第27页习题6.7题目和答案

 

 

原题需要使用数组编程实现，而这一章尚未学到数组，所以修改题目和答案如下所示。

学习指导第25页-第27页原题为：

6.7假设今年的工业产值为100万元，产值增长率为每年c%，请编程计算当c分别为6、8、10、12时工业产值分别过多少年可实现翻一番（即增加一倍）。

参考答案：将产值增长率存于数组c[i]中，分别初始化为6、8、10、12。用变量y表示产值翻番所需的年数，则各年所对应的产值s[i]为：

s[i] = s[i] * (1 + c[i]/100.0)

利用迭代法计算，直到s[i] >= 200时为止。当s[i] >= 200时，表示已实现产值翻番。然后，对下一个增长率c[i]计算实现产值翻番所需的年数，直到对所有的c[i]计算完毕。

#include <stdio.h>

#define  CURRENT  100 

#define  N        4

main()

{

int  i;

       int  year;                              /* 产值翻番所需年数 */

       int   growRate[N] = {6,8,10,12};     /* 工业产值的增长率 */

       double  output;                        /* 工业总产值 */

       for (i=0; i<N; i++)

       {

           output = CURRENT;              /* 当年产值为100万元 */

           for (year=0; output<2*CURRENT; year++)

           {

            output = output * (1 + growRate[i] / 100.0);           

           }

           printf("When grow rate is %d%%, the output can be doubled after

                    %d years.\n", growRate[i], year);                                                                       

       }     

}

#include <stdio.h>

#define  CURRENT  100 

#define  N        4

main()

{

int  i;

       int  year;                              /* 产值翻番所需年数 */

       int   growRate[N] = {6,8,10,12};     /* 工业产值的增长率 */

       double  output;                        /* 工业总产值 */

       for (i=0; i<N; i++)

       {

           year = 0;

output = CURRENT;                 /* 当年产值为100万元 */

           do

           {

            output = output * (1 + growRate[i] / 100.0);   

year++;   

           }while (output < 2*CURRENT) ;

       }     

}

修改为：

6.7已知今年的工业产值为100万元，产值增长率为从键盘输入，请编程计算工业产值过多少年可实现翻一番（即增加一倍）。

参考答案：用符号常量CURRENT表示今年的工业产值为100万元，用变量growRate表示产值增长率，用变量year表示产值翻番所需的年数，则计算年产值增长额的计算公式为：

output  = output * (1 + growRate)

利用迭代法循环计算，直到output >= 2*CURRENT时为止。当output >= 2*CURRENT时，表示已实现产值翻番。此时，循环被执行的次数year即为产值翻番所需的年数。
